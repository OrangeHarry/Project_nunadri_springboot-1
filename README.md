# <h2>🏠너나들이</h2>
  
<p>너나들이 란 같은 건물내의 이웃들끼리 소통할 수 있는 웹 커뮤니티 입니다.
  같은 건물 내에 사는 사람들을 중심으로 커뮤니티를 형성하여 유기적인 소통과 다양한 활동 등을 활성화시켜 작은 범위의 네트워킹을 형성하는 것을 목표로 하는 서비스 입니다.</p>
